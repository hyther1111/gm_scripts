# EMQ Chat Fix
A script that fix the chat being jumped to bottom randomly

# Install
Requires
- Tampermonkey / Greasemonkey
- Install link
https://github.com/hyther1111/emq-chat-fix/raw/refs/heads/main/emq-chat-fix.user.js

# What does this script do
- Stop the chat from being randomly jumps to bottom when you scroll up
- Only jumps the chat to bottom on new messages by yourself
- The chat stays still for all other messages (e.g. room settings/pauses/messages from other players)

# Possibly existing bugs (not tested yet so they could be not existed)
- Have to refresh the page when you create a room/start quiz/end quiz/return to room
